# Test API Folder

This folder contains tests for the public APIs of t8code.

## Subfolders

- [t8_fortran_interface/](t8_fortran_interface/README.md): Tests for the Fortran interface.

## Main Features

The tests in this folder verify the correctness and stability of the t8code APIs, ensuring that they can be reliably used by external applications.

## Main Entry Points

The tests are integrated into the main test suite and can be run using the `t8_gtest_main` executable after building the tests with CMake.
