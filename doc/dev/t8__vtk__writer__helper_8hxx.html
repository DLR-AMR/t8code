◆ t8_forest_vtk_point_to_element_ref_coords

const double t8_forest_vtk_point_to_element_ref_coords[T8_ECLASS_COUNT][T8_FOREST_VTK_QUADRATIC_ELEMENT_MAX_CORNERS][3]

Map vtk element corners to element reference coordinates.

The reference coordinates are defined in such a way, that the linear vtk corners are listed first and then the curved coords. This way, this array can be used for linear vtk elements as well as quadratic vtk elements.
