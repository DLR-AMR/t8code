Detailed Description

A C++ interface for the search functionality.

The user can define search and query callbacks to perform the search and query operations on the forest. Implementation details regarding the callback handling are given by https://stackoverflow.com/questions/2298242/callback-functions-in-c We decided for option 4, using std::function together with templates.
