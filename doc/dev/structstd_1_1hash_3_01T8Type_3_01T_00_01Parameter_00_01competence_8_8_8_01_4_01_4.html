Detailed Description

template<typename T, typename Parameter, template< typename > class... competence>
struct std::hash< T8Type< T, Parameter, competence... > >

Functor for hashing T8Type objects.

This struct defines a functor that computes the hash value of a T8Type object. It uses the std::hash function to generate the hash value based on the underlying type T of the T8Type object.

Template Parameters
TThe underlying type of the T8Type object.
ParameterAdditional parameters for the T8Type object.
competenceVariadic template parameters for additional competencies.
Note
This functor is enabled only if the T8Type object is hashable, as determined by the is_hashable member of the T8TypeImpl type.
