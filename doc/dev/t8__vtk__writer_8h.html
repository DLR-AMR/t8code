Translate a forest into a vtkUnstructuredGrid with respect to the given flags.

This function uses the vtk library. t8code must be configured with "--enable-vtk" in order to use it.

Parameters
[in]forestThe forest.
[in,out]unstructuredGrid
[in]write_treeidIf true, the global tree id is written for each element.
[in]write_mpirankIf true, the mpirank is written for each element.
[in]write_levelIf true, the refinement level is written for each element.
[in]write_element_idIf true, the global element id is written for each element.
[in]curved_flagIf true, write the elements as curved element types from vtk.
[in]write_ghostsIf true, write out ghost elements as well.
[in]num_dataNumber of user defined double valued data fields to write.
[in]dataArray of t8_vtk_data_field_t of length num_data providing the user defined per element data. If scalar and vector fields are used, all scalar fields must come first in the array.

Write the forest in .pvtu file format.

Writes one .vtu file per process and a meta .pvtu file. This function writes ASCII files and can be used when t8code is not configure with "--enable-vtk" and t8_forest_vtk_write_file_via_API is not available.

Parameters
[in]forestThe forest.
[in]fileprefixThe prefix of the output files.
[in]write_treeidIf true, the global tree id is written for each element.
[in]write_mpirankIf true, the mpirank is written for each element.
[in]write_levelIf true, the refinement level is written for each element.
[in]write_element_idIf true, the global element id is written for each element.
[in]write_ghostsIf true, each process additionally writes its ghost elements. For ghost element the treeid is -1.
[in]num_dataNumber of user defined double valued data fields to write.
[in]dataArray of t8_vtk_data_field_t of length num_data providing the used defined per element data. If scalar and vector fields are used, all scalar fields must come first in the array.
Returns
True if successful, false if not (process local).

Write the forest in .pvtu file format.

Writes one .vtu file per process and a meta .pvtu file. This function uses the vtk library. t8code must be configured with "--enable-vtk" in order to use it. Currently does not support pyramid elements.

Parameters
[in]forestThe forest.
[in]fileprefixThe prefix of the output files. The meta file will be named fileprefix.pvtu .
[in]write_treeidIf true, the global tree id is written for each element.
[in]write_mpirankIf true, the mpirank is written for each element.
[in]write_levelIf true, the refinement level is written for each element.
[in]write_element_idIf true, the global element id is written for each element.
[in]curved_flagIf true, write the elements as curved element types from vtk.
[in]write_ghostsIf true, write out ghost elements as well.
[in]num_dataNumber of user defined double valued data fields to write.
[in]dataArray of t8_vtk_data_field_t of length num_data providing the user defined per element data. If scalar and vector fields are used, all scalar fields must come first in the array.
Returns
True if successful, false if not (process local).
Note
If t8code was not configured with vtk, use t8_forest_vtk_write_file
