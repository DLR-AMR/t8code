Construct a new vtk writer object.

All parameters are set to false by default. By default no data is used and num_data is set to zero. A default fileprefix is NOT given.

Parameters
write_treeidTrue, if we want to write the tree id of every element.
write_mpirankTrue, if we want to write the mpirankof every element.
write_levelTrue, if we want to write the level of every element. Uses level 0 if used for a cmesh.
write_element_idTrue, if we want to write the element id of every element. Ignored if used for a cmesh.
write_ghostsTrue, if we want to write the ghost elements, too.
curved_flagTrue, if we want to use quadratic vtk cells. Uses the geometry of the grid to evaluate points between corners.
fileprefixThe prefix of the output-file.
num_dataThe number of data-fields to print.
dataThe data to use.
commThe communicator for parallel output.
