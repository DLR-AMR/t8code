Construct a hypercube forest from one primitive tree class.

Parameters
[in]eclassThis element class determines the dimension of the cube.
[in]commThe mpi communicator to be used.
[in]boundaryThe vertices, that define the hypercube boundary.
[in]polygons_xThe number of polygons along the x-axis.
[in]polygons_yThe number of polygons along the y-axis. Only required if eclass is 2D or 3D.
[in]polygons_zThe number of polygons along the z-axis. Only required if eclass is 3D.
Returns
A committed t8_cmesh structure with polygons_x * polygons_z * polygons_y many sub-hypercubes of class eclass.
Note
boundary must point to an array with 3*8 (3D), 3*4 (2D), 3*2 (1D), or 3 (0D) entries.
Every sub-hypercube contains different number of trees depending on eclass.
If eclass == T8_ECLASS_VERTEX, _LINE, _QUAD or _HEX every sub-hypercube contains one tree, if _TRIANGLE or _PRISM two trees and if _TET six trees. This is done in the same way as in
